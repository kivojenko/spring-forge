# Version 0.1.15 (2026-06-01)

### Features
- **Nested and Association Filtering**: Introduced `targetField` in `@FilterField` annotation. This allows filtering by properties of associated entities (e.g., filtering `Product` by `category.name`) or nested paths.
- **Custom Filter Parameter Names**: Added `name` attribute to `@FilterField` allowing customization of query parameter names in REST endpoints and Filter DTOs, while preserving mapping to original entity fields.

### Improvements
- **Type-safe Persistence Hooks**: Enhanced `ForgePersistenceAspect` with `entityType()` requirement, ensuring that AOP persistence hooks are only executed for the intended entity type. Improved handling of association-based hooks (e.g., `beforeAdd`) to correctly identify the target entity type even when IDs are passed to service methods.
- **Repository Method Safeguard**: Added protection against method name collisions when both `HasName` and `@FilterField` with the same name are present.
- **Improved Test Infrastructure**: Migrated repository-level integration tests to controller-level to ensure full coverage of the generated REST API and its interaction with the underlying services and repositories.
