# Version 0.1.13 (2026-02-17)

### Changes
- **Advanced String Filtering**: Added support for advanced string filtering via `@StringFilterField` annotation.
- **Multiple String Match Modes**: Supported various string match modes:
  - `EQUALS`
  - `EQUALS_IGNORE_CASE`
  - `CONTAINS`
  - `CONTAINS_IGNORE_CASE`
  - `STARTS_WITH`
  - `ENDS_WITH`
- **Configurable Page Size**: Added `pageSize` configuration to `SpringForgeConfig` for better control over pagination.
