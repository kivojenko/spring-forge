# Version 0.1.14 (2026-04-07)

### Changes
- **Generalized FilterField**: Refactored `FilterField` to be more generic and support various types of filters.
- **Range Filtering Support**: Added support for range filtering via new annotation `@RangeFilterField`
- **Javadoc Generation**: Improved generated code quality by adding JavaDoc to generated methods and fields.
- **Improved Filtering Tests**: Added comprehensive tests for various filtering scenarios.

### Bug Fixes
- **File Reading**: Fixed an issue with file reading in certain environments.
