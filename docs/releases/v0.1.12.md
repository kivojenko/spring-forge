# Version 0.1.12 (2026-02-17)

### Changes
- **Improved HTTP status codes**: Generated endpoints now return more appropriate HTTP status codes:
  - `201 Created` for create/add/link operations.
  - `204 No Content` for delete/unlink operations.
- **Refactoring of `ClassNameUtils`**: Converted `ClassNameUtils` from a class to an interface for better usage and cleaner implementation.
- **Updated Tests**: Reflected new HTTP status codes in `AuthorControllerTest`, `BookControllerTest`, and `CategoryControllerTest`.
